Guide to creating custom form control using controlValueAccessor

this project contains an overview/guide to create and use custom form controls by using some apis that angular provides us in the Forms Module.
these custome controls can be used to create reusable forms and also help you understand how to create your own For example: star rating, quantity(increase decrease) and many more


Angular internally use control value accessor for some controls such as

CheckboxControlValueAccessor DefaultValueAccessor NumberValueAccessor RadioControlValueAccessor RangeValueAccessor SelectControlValueAccessor SelectMultipleControlValueAccessor
